4. Modularity

The modularity on this site is pretty simple, it consists of a banner stretching over the top, with the different home, menu, about and location sites. Beneath this there are multiple divs. A good example is the menu.html page, where there is a body, which contains multiple divs, like the navbar, and menu group div, containing the different dishes available. On the right side there is a your orer div, which contains the items youre planning on buying. Layered above this, is the overlay which pops up if clicked on one of the menu items.
